本管理システム

作成した経緯

家に書籍が増えてきて何とかして管理しようと思ったときにこのシステムを作成しようと考えました。

使用した技術

  • HTML
  • CSS
  • PHP(Laravel)
  • Github
  • Heroku

心がけたこと

ログイン機能の実装、本の追加、削除、更新の機能そしてMysqlを用いたデータベース管理を行いました。その中で自分が使用する際にもできるだけ使いやすくするようにしました。

苦労した点

PHP、sqlの知識がほぼ0の状態から始めたので基本的にネットを頼りにして一つ一つ機能を実装しました。今までプログラミングを行う際は1つか2つのファイルを使用していました。しかしLaravelを用いてwebアプリケーションを制作していく中で最初にプロジェクトを作成しました。すると大量のファイルが出てきてどこに何があるかわからずそこから調べる事に苦労しました。
